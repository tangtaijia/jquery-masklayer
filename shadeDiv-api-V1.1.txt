标题：遮罩层1.1
api
	method：init-初始化
		    param
		    	box 被遮罩的元素 默认为document
			    msg 展示出的 加载提示 默认为'waiting' 如 加载中，loading
			    size 尺寸对象
				    width 大小 可以数字指定像素或百分比 默认为'100%'
				    height 高度 可以数字指定像素或百分比 默认为'100%'
			    secTohide 几秒后隐藏  默认为'none'
	             example：
			 1(全屏遮罩)
	          shadeDiv.init({
			    msg : 'loading...',
			    secTohide : 2
			  });
			 2(元素遮罩)
			  shadeDiv.init({
	            box:".hotel_pic",
	            msg : '请稍等...'
	          });
				
			hide-隐藏
		   		example：
					shadeDiv.hide();
引入
	<link href="js/shadeDiv/jquery.shadeDiv.css" rel="stylesheet" type="text/css" />
	<script src="js/shadeDiv/jquery.shadeDiv.min.js" type="text/javascript" language="javascript"></script>
